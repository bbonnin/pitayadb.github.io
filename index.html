Pitaya DB

A NoSQL database running in NodeJS

storage

NoSQL DB

PitayaDB is a key/value database : you can store any kind of documents in a collection and get them using their key.

memory

In memory / On disk

PitayaDB can store your documents in memory or on disk, depending on how you love your data.

computer

Embedded and server modes

PitayaDB can be embedded in your NodeJS application or can run as a standalone application.

language

Web API and UI

PitayaDB provides a simple REST API to access your data and a UI to manage and supervise all the stuff.
